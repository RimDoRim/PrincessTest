# 👑 나는 어떤 공주일까?

MZ 트렌드 감수성을 테스트하는 심리 테스트 웹앱.

이름을 입력하고 8개 질문에 1~5점으로 답하면 총점에 따라 5가지 등급 중 하나를 보여줍니다.

## 등급

| 등급 | 점수 |
|------|------|
| 👑 진짜 공주님 | 33점 이상 |
| 🎀 무언가 부족한 공주님 | 25~32점 |
| 🌿 평민 | 17~24점 |
| 🪴 서민 | 8~16점 |
| 🧹 하녀 | 8점 미만 |

## 기술 스택

- Vue 3 (Composition API)
- Vite
- Pinia
- Vue Router 4
- Bootstrap 5
- Font Awesome 6

## 로컬 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
