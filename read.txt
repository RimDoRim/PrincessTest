Vue3 + Vite 기반 "공주 테스트" 웹사이트를 만들어줘.

## 서비스 개요
사용자 이름을 입력받고, 최근 유행하는 트렌드에 관한 8개 질문에
1~5점으로 답변하면 총점에 따라 5단계 등급을 보여주는 심리 테스트.
일부 문항은 역점수 처리가 필요하다.

⭐ 핵심 이스터에그: 이름이 정확히 "김민정"이면
   점수와 무관하게 무조건 "진짜 공주님" 등급이 나와야 한다.
   이 로직은 절대 우회되거나 노출되면 안 됨.

## 기술 스택
- Vue 3 (Composition API, <script setup> 방식)
- Vite (빌드 도구)
- JavaScript ES6+
- Bootstrap 5 (CDN)
- Font Awesome 6 (CDN)
- Pinia (상태 관리)
- Vue Router 4

## 점수 계산 방식 (중요 - 반드시 이 로직 사용)
- isReverse: false → 선택값 그대로 점수 (1~5)
- isReverse: true  → 역산 처리: 실제점수 = 6 - 선택값
  (예: 5 선택 → 1점, 1 선택 → 5점)
- totalScore = 8개 문항 실제점수 합산 (최소 8점, 최대 40점)

## 등급 판별 로직 (반드시 이 코드 그대로 사용)
const getResultGrade = computed(() => {
  // 1순위: 이스터에그 (점수 무시, 무조건 진짜 공주님)
  if (userName.value === '김민정') return '진짜 공주님';
  // 2순위: 점수에 따른 등급 판별
  const score = totalScore.value;
  if (score >= 43) return '진짜 공주님';       // 최대 40점으로 물리적 불가 (이스터에그 전용 구간)
  if (score >= 33) return '무언가 부족한 공주님';
  if (score >= 25) return '평민';
  if (score >= 17) return '서민';
  return '하녀';
});

## 필수 구현 내용

### stores/quizStore.js (Pinia Composition API 방식)
- userName (string) 상태
- answers (배열, 8개 문항의 선택값 저장)
- totalScore: computed → isReverse 여부에 따라 점수 계산
  questions[i].isReverse ? 6 - answers[i] : answers[i] 로 합산
- getResultGrade: computed → 위 등급 판별 로직 그대로 사용
- computeResult(): getResultGrade 값으로 resultDescriptions 조회 → result 에 저장
- result 상태: { grade, description, image, emoji, color }
- resetQuiz(): userName, answers, result 초기화

### Home.vue (1단계: 이름 입력)
- 전체 화면 세로 중앙 정렬
- 👑 + "나는 어떤 공주일까?" 타이틀 (큰 폰트)
- 부제목: "솔직하게 답해야 진짜 결과가 나옵니다 👀"
- 이름 입력 input (placeholder: "이름을 입력하세요", 둥근 테두리)
- "테스트 시작하기" 버튼 → /quiz 이동
- trim() 후 빈 문자열이면 버튼 비활성화 + 회색 처리
- quizStore.userName에 입력값 저장 후 라우팅

### Quiz.vue (2단계: 질문)
- 한 번에 질문 1개씩 (슬라이드 방식)
- 상단: Bootstrap progress bar + "3 / 8" 텍스트
- QuestionCard 컴포넌트에 현재 질문 text 전달
- 선택지 5개 버튼:
  1️⃣ 전혀 아니다
  2️⃣ 아니다
  3️⃣ 보통이다
  4️⃣ 그렇다
  5️⃣ 매우 그렇다
- 선택 시 answers에 선택값(1~5) push → 자동으로 다음 질문 이동
- 질문 전환 시 Vue <Transition> 페이드 효과
- 8번 완료 시 computeResult() 호출 → /result 이동
- 브라우저 뒤로가기 방지 (popstate 이벤트 차단)
- userName 없으면 /home 리다이렉트

### Result.vue (3단계: 결과)
- ResultCard 컴포넌트 사용
- 등급 이미지 중앙 표시 (width: 200px, border-radius: 50%)
  → onerror 시 emoji 폴백
- 등급명 크게 표시 (예: 👑 진짜 공주님)
- 등급별 설명 텍스트
- "다시 하기" 버튼 → resetQuiz() 후 /home
- "결과 공유하기" 버튼: 클립보드 복사
  형식: "[이름]님의 공주 테스트 결과: [등급명]!\n[설명]\n👉 테스트 해보기: (URL)"
  복사 완료 시 버튼 텍스트 "✅ 복사됨!" 으로 1.5초 후 원복
- "진짜 공주님" 카드: 골드(#FFD700) 테두리 2px + 별 반짝이 CSS 애니메이션
